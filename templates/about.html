What do I do?

I work on many different messaging platforms along with this web app. The commands throughout the different platforms are the same. Just type something in, and I'll respond if I have something to say about it! Up ahead are some common commands. Text in brackets ([]) are things you must type in.

  • Weather: Get the weather in [city]
  • Expressions: [calculate|calc] [expression], e.g. calc 2 * 3 ^ 5
  • Nickname: Call me [nickName]
  • Current time and date: Time
  • Record Like: I like [something]
  • Remove Like: I don't like [something]
  • List Likes: Likes
  • Gravatar: Gravatar [email]

There are many more things I respond to, so have fun and try it out!

How does it work?

When I receive a new message from any app, the first thing I do is check to see if the user is registered. If not, I'll record his or her information so that I can remember likes, nicknames, and more! After making sure I know who the user is, I'll look through a list of responses to see which response is the best. Once I find the best option, I'll send it back! If I can't find something that seems reasonable, I won't send anything back.

What about reminders? I actually check once per second to see if any reminders are ready to be sent out. If I see one that is ready, I'll look up the user and chat associated with the reminder and send it out!
